> 项目：考勤填报系统  
> 版本：`attendance-report v0.1.0`  
> 测试日期：____年__月__日  
> 测试人员：________

## 1. 测试范围

本次测试覆盖以下模块：

- 登录与会话
- 员工端：每日填报、历史记录、月度提交、个人导出
- 管理端：员工列表、提交情况、按日编辑、全员导出
- 导出：CSV / Excel / PDF（中文内容）
- 权限控制：员工与管理员角色隔离

## 2. 测试环境

- OS：macOS
- Node.js：20+
- 数据库：SQLite（`prisma/dev.db`）
- 启动方式：

```bash
cd /Users/liyifan/Desktop/attendance-report
npm install
npx prisma db push
npx prisma db seed
npm run dev
```

- 访问地址：`http://localhost:3000`（若 3000 被占用则按终端实际端口）
- 测试账号：
  - 管理员：`admin@company.local / demo123`
  - 员工：`yamada@company.local / demo123`
